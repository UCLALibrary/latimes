About LA Times Search Tool

The Los Angeles Times Photographic Index Search tool allows users to search an index of images contained in UCLA's LA Times Photographic Archive. This tool does not provide users with digital surrogates of these images. Instead it provides an index of these images, which can be viewed in person at UCLA's Special Collections. Sections

Keyword Search

Keyword search allows for searching of a keyword. It will search against all fields a record (Subject Name, Subject Description, Photo Description, Box Number, Quantity, Negative Number, Month, Day and Year). It also provides the user with facets of Box Number, Negative Number, and Year.
NOTE: When searching for items with apostrophes (') search with space instead of apostrophe

Advanced Search

Advanced search allows for searching of a keyword, year and box number. The keyword will search against all fields a record (Subject Name, Subject Description, Photo Description, Box Number, Quantity, Negative Number, Month, Day and Year). It also provides the user with facets of Negative Number, Subject Name and Subject Description.

Browse

Browse allows users to view all items in the LA Times photographic index. This browse functionality is aided sorting by clicking on headings. It also allows for these records to be narrowed down by a search query, which searches against all fields a record (Subject Name, Subject Description, Photo Description, Box Number, Quantity, Negative Number, Month, Day and Year). {% endblock %}
